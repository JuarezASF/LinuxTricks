No arquivo vimrc digite:

set global spell spelllang=pt

set spell 


Para instalar o dicionário:

-remova dos plugins qualquer coisa que tenha tentado antes
-abra o vim e digite:
setlocal spell spelllang=pt

-o próprio vim irá baixar o dicionário e instalar

*funcionou aqui no vortex

usando o spell check:

digite (em modo normal) [s para ir para a próxima palavra errada.

com o cursos antes da palavra digite z= para ver uma lista de palavras sugeridas

digite o número da palavra na lista para que substituição seja feita
